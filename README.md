# scsa-issue-tracker

SCSA issue tracker team project.

## Tech Stack

- Backend: Java 17, Spring Boot 3.5.14, Gradle
- Frontend: TBD
- Database: TBD

## Repository Structure

```text
scsa-issue-tracker/
├── backend/   # Spring Boot backend API
├── frontend/  # Frontend application
├── docs/      # ERD, API specs, requirements
├── README.md
└── .gitignore
```

## Backend

```bash
cd backend
./gradlew bootRun
```

On Windows:

```bash
cd backend
gradlew.bat bootRun
```
